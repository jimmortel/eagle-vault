# 🦅 Eagle Vault

**The first collaborative golden vault on Solana, built entirely from a smartphone.**

## 💡 Concept
Eagle Vault is a social experiment and a micro-lottery. 
- **Goal:** Reach 0.1 SOL.
- **Participation:** Anyone can add 0.005 SOL to the vault.
- **Reward:** When the goal is reached, **50% of the total** is automatically sent to the **last contributor**. The other 50% supports my mobile-only development journey.

## 🛠️ Tech Stack
- **Frontend:** HTML5 / CSS3 (Cyber-gold theme)
- **Blockchain:** Solana Web3.js
- **Backend:** Vercel Serverless Functions
- **Database:** Vercel KV (Key-Value)
- **Built on:** Android (Mobile-only dev)

## 🚀 How to participate
1. Connect your Solana Wallet (Phantom/Solflare).
2. Click "Unlock".
3. Become the "Last Keyholder" to win the jackpot.
